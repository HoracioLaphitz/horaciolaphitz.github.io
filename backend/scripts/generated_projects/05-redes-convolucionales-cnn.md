## Descripción del Proyecto

Implementación de redes neuronales convolucionales profundas para visión por computadora. Incluye arquitecturas CNN avanzadas, técnicas de Transfer Learning y Fine-tuning, aumentación de datos y evaluación rigurosa de modelos visuales.

### Características Destacadas

- Arquitecturas CNN
    - Transfer Learning
    - Aumentación de datos
    - Evaluación de modelos visuales

### Tecnologías Utilizadas

- Python
- TensorFlow
- Keras
- OpenCV
