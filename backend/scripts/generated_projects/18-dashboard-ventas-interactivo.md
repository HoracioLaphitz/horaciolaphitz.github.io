## Descripción del Proyecto

Dashboard interactivo de análisis comercial. Incluye visualización de datos comerciales en tiempo real, KPI's interactivos y actualizados, análisis de tendencias por período y análisis de rentabilidad por producto.

### Características Destacadas

- Datos en tiempo real
    - KPI's interactivos
    - Tendencias por período
    - Rentabilidad por producto

### Tecnologías Utilizadas

- Python
- Tableau
- SQL
