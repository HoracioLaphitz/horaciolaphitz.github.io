## Descripción del Proyecto

Proyecto completo de machine learning aplicado a predicción de precios de casas. Incluye análisis exploratorio de datos (EDA) completo, desarrollo e implementación de modelos de regresión lineal, técnicas de normalización y escalado de datos, y validación rigurosa de modelos predictivos.

### Características Destacadas

- Análisis exploratorio de datos (EDA) completo
    - Modelos de regresión lineal
    - Normalización y escalado de datos
    - Validación y evaluación de modelos

### Tecnologías Utilizadas

- Python
- Scikit-learn
- Pandas
- NumPy
