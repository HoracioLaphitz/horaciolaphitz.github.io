## Descripción del Proyecto

Implementación de Pandas Profiling para análisis automático de datos. Incluye generación automática de reportes EDA completos, análisis de calidad de datos, detección de valores faltantes e identificación de outliers y anomalías.

### Características Destacadas

- Reportes EDA automáticos
    - Análisis de calidad
    - Detección de faltantes
    - Identificación de anomalías

### Tecnologías Utilizadas

- Python
- Pandas Profiling
- Pandas
