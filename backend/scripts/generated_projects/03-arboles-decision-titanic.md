## Descripción del Proyecto

Proyecto de machine learning usando árboles de decisión sobre el famoso dataset del Titanic. Incluye desarrollo y optimización de árboles de decisión, análisis de importancia de variables, técnicas de poda y optimización, y visualización de estructuras de árbol complejas.

### Características Destacadas

- Árboles de decisión
    - Análisis de importancia de variables
    - Poda y optimización
    - Visualización de árboles

### Tecnologías Utilizadas

- Python
- Scikit-learn
- Pandas
- Graphviz
