## Descripción del Proyecto

Proyecto integral de procesamiento ETL. Incluye extracción de múltiples fuentes de datos, transformación y limpieza rigurosa, carga en sistemas de análisis y desarrollo de pipelines de datos automatizados y robustos.

### Características Destacadas

- Extracción de múltiples fuentes
    - Transformación y limpieza
    - Carga en sistemas
    - Pipelines automatizados

### Tecnologías Utilizadas

- Python
- Pandas
- SQL
- n8n
