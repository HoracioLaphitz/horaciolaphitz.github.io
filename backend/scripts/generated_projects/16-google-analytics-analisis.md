## Descripción del Proyecto

Proyecto de análisis de Google Analytics. Incluye extracción de datos de Google Analytics, análisis de tráfico web, segmentación de usuarios y análisis profundo de comportamiento del usuario.

### Características Destacadas

- Extracción de GA
    - Análisis de tráfico
    - Segmentación de usuarios
    - Comportamiento del usuario

### Tecnologías Utilizadas

- Python
- Google Analytics API
- Pandas
