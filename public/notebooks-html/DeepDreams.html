ALGORITMO DEEP DREAM

Este es un script desarrollado con python el cual utiliza un modelo de aprendizaje automatico previamente entrenado(imagenet) para poder analizar y visualizar como los modelos de aprendizaje automatico visualizan el mundo

El mismo utiliza tecnicas como el Ascenso de Gradiente para deformar las imagenes y poder tener una nocion de como se imagina el mundo una IA.

#OBJETIVO:

  • Seleccionar una capa e intentar maximiza la perdida que son las activaciones generadas por la capa de interés.

  • Se Pueden seleccionar cualquier capa que se quiera , las capas tempranas generan caracteristicas simples como Bordes, y las capas profundas generan caracteristicas mas complejas como toda una cara(de persona , un auto o un árbol)

  • La Red de inicio tiene multiples capas concatenadas llamadas "mixtas"

Calculo directo de la Funcion de Perdida

El objetivo de la funcion es seleccionar una capa e intentar maximizar la perdida

Se pueden Seleccionar cualquier capa que se requiera.

La mayoria de las veces se quiere minizar la perdida o "error" a traves del Descenso de Gradiente

  1. Utilizada para el calculo de la Pérdida
  2. Funciona alimentando la imagen de entrada a traves de la red y generando activaciones
  3. Despues otinene el promedio y suma de esos resultados
def calculo_perdida(imagen,modelo):
  img_batch = tf.expand_dims(imagen, axis=0) # convertir el formato de Lotes/ Batch
  layer_activations = modelo(img_batch) #Corriendo el Modelo
  print(f"Valores de Activacion (LAYER OUTPUT) : {layer_activations}")

  perdidas= [] #acumulador de las perdidas
  for act in layer_activations:
    perdida = tf.math.reduce_mean(act)# calcula la media de las perdidas
    perdidas.append(perdida)
  print(f"Perdidas (Multiples capas de Activacion) : {perdidas}")
  #print(f"Forma de Perdida (Multiples capas de Activacion) : {np.shape(perdidas)}")
  #print(f"Suma de Todas las Perdidas (De todas las Capas Seleccionadas) : {tf.reduce_sum(perdidas)}")
  return tf.reduce_sum(perdidas) # Calculo de la Suma

def deepdream(modelo,imagen,tamano_pasos):
    with tf.GradientTape() as tape:
      # se necesitan gradientes relativos a las imagenes 'img'
      # 'GradientTape() solo mira a 'tf.Variable's por defecto'
      tape.watch(imagen)
      perdida = calculo_perdida(imagen,modelo) #llamado a la funcion de perdida
    # Calculo del gradiente
    gradiente = tape.gradient(perdida, imagen)

    print(f"Gradiente : {gradiente}")
